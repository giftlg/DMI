Privacy Policy

August 01, 2018

This page informs you of our policies regarding the use of personal data when you use "Finder".

Information Collection And Use

The application needs the following types of information to work properly.

Required permissions:

  • ACCESS_FINE_LOCATION and ACCESS_COARSE_LOCATION
  • RECEIVE_SMS
  • READ_SMS
  • SEND_SMS
  • INTERNET
  • ACCESS_NETWORK_STATE
  • CHANGE_WIFI_STATE
  • ACCESS_WIFI_STATE
  • READ_PHONE_STATE
  • WRITE_EXTERNAL_STORAGE
  • READ_EXTERNAL_STORAGE

Use of Data

"Finder" uses these permissions for the following purposes:

  • LOCATION - to send the location of the mobile phone and display your location on the built-in map
  • RECEIVE_SMS - to detect incoming SMS commands
  • SEND_SMS - to answer incoming commands and send requests
  • READ_SMS - to have the ability to open received data SMS messages in the case of the broadcast receiver not working
  • INTERNET - to use the "Google geolocation API" (when wifi and network data are received) and built-in Google map
  • ACCESS_NETWORK_STATE - to check network state before the wifi request is sent, displaying a warning if there is no network connectivity
  • CHANGE_WIFI_STATE - to enable wifi to scan for wifi networks (during wifi request)
  • ACCESS_WIFI_STATE - to determine the wifi state before scanning, to leave it in the same condition
  • READ_PHONE_STATE - to scan mobile networks
  • WRITE_EXTERNAL_STORAGE,READ_EXTERNAL_STORAGE - to allow the usage of the built-in Google map

Transfer Of Data

The application does not send any information to the developers. The collected data may only be sent to the phone numbers you specify, via an SMS message. To determine the coordinates using network data the "Google geolocation API" is used.
